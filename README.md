## Overview

Fit-AI leverages AI technology to offer personalized fitness advice, track progress, and manage workout and diet plans. This project integrates several free KendoReact components to enhance the user interface and experience.

## KendoReact Experience

### Free KendoReact Components Used

1. **Chat**
   ```typescript
   // src/pages/Chat.tsx
   import { Chat } from "@progress/kendo-react-conversational-ui";
   ```

2. **Button**
   ```typescript
   // src/App.tsx
   import { Button } from "@progress/kendo-react-buttons";
   ```

3. **Input**
   ```typescript
   // src/components/Form.tsx
   import { Input } from "@progress/kendo-react-inputs";
   ```

4. **Grid with GridColumn**
   ```typescript
   // src/pages/Stats.tsx
   import { Grid, GridColumn } from "@progress/kendo-react-grid";
   ```

5. **TabStrip with TabStripTab**
   ```typescript
   // src/components/TabComponent.tsx
   import { TabStrip, TabStripTab } from "@progress/kendo-react-layout";
   ```

6. **Card with CardTitle, CardBody, CardActions**
   ```typescript
   // src/components/CardComponent.tsx
   import { Card, CardTitle, CardBody, CardActions } from "@progress/kendo-react-layout";
   ```

7. **Form**
   ```typescript
   // src/components/CustomForm.tsx
   import { Form, FormElement, Field } from "@progress/kendo-react-form";
   ```

8. **DropDownList**
   ```typescript
   // src/components/DropDownListComponent.tsx
   import { DropDownList } from "@progress/kendo-react-dropdowns";
   ```

9. **Chart with various sub-components**
   ```typescript
   // src/pages/Stats.tsx
   import { Chart } from "@progress/kendo-react-charts";
   ```

10. **ProgressBar**
    ```typescript
    // src/components/ProgressBarComponent.tsx
    import { ProgressBar } from "@progress/kendo-react-progressbars";
    ```

11. **Switch**
    ```typescript
    // src/components/SwitchComponent.tsx
    import { Switch } from "@progress/kendo-react-inputs";
    ```

12. **Dialog with DialogActionsBar**
    ```typescript
    // src/components/DialogComponent.tsx
    import { Dialog, DialogActionsBar } from "@progress/kendo-react-dialogs";
    ```

### Aim to Impress

Fit-AI integrates GenAI technology to provide real-time AI-driven fitness recommendations and insights.

### Delightfully Designed

Fit-AI utilizes the KendoUI Figma Kits and Progress ThemeBuilder to create a visually appealing and user-friendly interface. The design process involved regular feedback loops and iterations to refine the UI/UX.

## Installation

To get started with Fit-AI, clone the repository and install the dependencies:

```bash
git clone https://github.com/sakshamm0507/Fit-AI.git
cd Fit-AI
npm install
```

## Usage

Run the development server:

```bash
npm start
```

Open your browser and navigate to `http://localhost:3000` to see Fit-AI in action.

## Features

- AI-driven fitness recommendations
- Real-time chat with the AI assistant
- Fitness progress tracking and analytics
- Customizable workout and diet plans
